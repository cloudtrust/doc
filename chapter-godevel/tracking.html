Tracking

Tracking

Sentry is an open source error tracking system. It helps monitoring crashes and errors in real time.

Sentry

By default, the error tracking is deactivated. To set it up, add the sentry-dsn to the configuration file.

Note: To obtain the sentry Data Source Name (DSN) you need to set up a project in Sentry (see documentation).

Key Description Default value
sentry-dsn Sentry Data Source Name ""
